# Embroidery Design Visualizer

A full-stack JavaScript application for visualizing embroidery designs on model pictures with customizable dress attributes.

## Features

- Upload and manage embroidery designs
- Customize dress attributes (color, cloth type, style)
- Place designs on specific dress parts
- Support for multiple dress categories (bridal, casual, traditional, party wear)
- Real-time preview and editing
- Export final designs

## Tech Stack

**Frontend:**
- React with Vite
- Fabric.js for canvas manipulation
- Tailwind CSS for styling
- Axios for API calls

**Backend:**
- Node.js with Express
- MongoDB with Mongoose
- Sharp for image processing
- Multer for file uploads
- Cloudinary for cloud storage
